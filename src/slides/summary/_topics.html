What are we going to
talk about today?

Past

  • Float and Clear: The original layout techniques that led to clearfix hacks.
  • Table Layouts: Using tables for layout, leading to accessibility issues.
  • CSS Sprites: A method for reducing HTTP requests, but complex to maintain.
  • Creative Hacks: Browser-specific hacks and workarounds that were fragile and hard to debug.

Baseline

Since 2023 this project has the intent to simplify the way we approach web design and development

Present

  • Layout: Flexbox, Grid, Multi-column, Subgrid, Aspect Ratio, Container Queries.
  • Functions: calc(), min(), max(), clamp(), math functions (sin(), cos(), tan()), color functions (rgb(), hsl(), lab()).
  • Effects: Filters (blur, brightness, contrast), Masks, Clip-path, Backdrop-filter.
  • Interactivity: :hover, :focus, :active, :focus-within, :has(), Pointer Events, Media Queries.
  • Typography: Custom Properties, Variable Fonts, Font Display, Text Decoration, Text Overflow.
  • At-rules: @media, @supports, @layer, @import, @font-face, @keyframes.
  • Animations: Transitions, Keyframe Animations, Scroll-driven Animations (Scroll Timeline).

Future

  • Custom Functions: User-defined functions for CSS.
  • If() Function: Conditional logic in styles.
  • Anchor Positioning: Improved handling of anchor links.
  • Invokers: New way to trigger styles and animations.
