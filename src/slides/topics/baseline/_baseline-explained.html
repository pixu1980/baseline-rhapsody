What is Baseline?

Baseline is a standard that indicates when web platform features become reliably available across major browsers.

The 3 Baseline States

  • Widely: Features that are fully supported across all modern browsers.
  • Newly: Features that have recently been implemented and may not be fully supported yet.
  • Limited: Features that are only available in specific browsers or under certain conditions.

Checking Baseline Status

  1. Visit web.dev/baseline
  2. Check MDN browser compatibility tables
  3. Look for Baseline badges in documentation
  4. Use @supports for feature detection
/* Feature detection example */
@supports (container-type: inline-size) {
  .card {
    container-type: inline-size;
  }
}

/* Fallback for browsers without support */
@supports not (container-type: inline-size) {
  .card {
    /* Use media queries instead */
  }
}
