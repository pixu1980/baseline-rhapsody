scrollbar-color

Simple Scrollbar Theming

.container {
  /* Firefox only (for now) */
  scrollbar-color: #667eea #e0e7ff;
  /*              thumb    track */
  
  scrollbar-width: thin; /* ‚úÖ Widely available */
}

/* Still need vendor prefixes for webkit */
.container::-webkit-scrollbar {
  width: 8px;
}

.container::-webkit-scrollbar-thumb {
  background: #667eea;
  border-radius: 4px;
}

.container::-webkit-scrollbar-track {
  background: #e0e7ff;
}

‚ùå Not in Baseline

Firefox 64+ - Full support
Chrome/Safari - Not supported, use ::-webkit-scrollbar

üé® Cross-Browser Approach

/* Progressive enhancement */
.custom-scroll {
  scrollbar-width: thin;
  scrollbar-gutter: stable;
}

/* Firefox */
@supports (scrollbar-color: auto) {
  .custom-scroll {
    scrollbar-color: #667eea #e0e7ff;
  }
}

/* Webkit browsers */
@supports selector(::-webkit-scrollbar) {
  .custom-scroll::-webkit-scrollbar {
    width: 8px;
  }
  .custom-scroll::-webkit-scrollbar-thumb {
    background: #667eea;
  }
  .custom-scroll::-webkit-scrollbar-track {
    background: #e0e7ff;
  }
}
