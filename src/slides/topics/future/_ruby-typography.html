What is Ruby Text?

Ruby text is small annotations above (or beside) base text, commonly used in East Asian typography for pronunciation guides, translations, or semantic annotations.

ruby-position

Position Control

ruby {
  ruby-position: over;  /* Default - above (horizontal) */
  ruby-position: under; /* Below */
  ruby-position: inter-character; /* Between chars (rare) */
}

ruby-align

Alignment Options

ruby {

  ruby-align: space-between; /* Distribute evenly */
  ruby-align: space-around;  /* Space around */
