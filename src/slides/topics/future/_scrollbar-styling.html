scrollbar-gutter

Reserve Space for Scrollbar

/* Prevents layout shift when content grows */
.container {
  scrollbar-gutter: stable;
  /* Always reserves space for scrollbar
     Even when content doesn't scroll */
}

/* Reserve space on both sides */
.centered-content {
  scrollbar-gutter: stable both-edges;
  /* Useful for centered content */
}

Combining Both Properties

Clean, Stable Scrolling

.content-area {
  /* Thin scrollbar */
  scrollbar-width: thin;
  
  /* Reserve space to prevent layout shift */
  scrollbar-gutter: stable;
  
  /* Clean, consistent layout */
  overflow-y: auto;
  max-height: 500px;
}

Custom Scrollbar (Firefox-only)

/* Firefox supports scrollbar-color */
.custom-scrollbar {
  scrollbar-width: thin;
  scrollbar-color: #667eea #e0e7ff;
  /*              thumb    track */
}

/* For webkit browsers, still need vendor prefixes */
.custom-scrollbar::-webkit-scrollbar {
  width: 8px;
}

.custom-scrollbar::-webkit-scrollbar-thumb {
  background: #667eea;
  border-radius: 4px;
}

.custom-scrollbar::-webkit-scrollbar-track {
  background: #e0e7ff;
}

üí° Use Cases

  • Mobile-like thin scrollbars on desktop
  • Hide scrollbars for cleaner UI (e.g., image carousels)
  • Prevent layout shift in dynamic content areas
  • Consistent width for centered layouts
