Animation Timeline ✅ Widely Available

/* Control animation progression */
.element {
  animation: slide 1s linear;
  animation-timeline: scroll(root);
  animation-range: 0% 100%;
}

/* View-based timeline */
.card {
  animation: fade-in 1s;
  animation-timeline: view();
  /* Start when entering viewport */
  animation-range: entry 0% cover 50%;
}

/* Named timeline */
@scroll-timeline my-scroller {
  source: selector(#container);
  orientation: block;
}

Timeline types:
• scroll() - Page/element scroll
• view() - Element visibility
• Custom timelines
