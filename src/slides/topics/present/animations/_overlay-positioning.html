Control the top layer

What is the Top Layer?

<!-- Elements that render in top layer -->

<!-- Fullscreen API -->
<div id="fullscreen">
  Fullscreen content
</div>

<!-- Dialog element -->
<dialog id="modal">
  Modal content
</dialog>

<!-- Popover -->
<div popover id="menu">
  Popover content
</div>
/* Top layer characteristics */
dialog[open] {
  /* Automatically in top layer */
  /* Renders above ALL other content */
  /* Ignores z-index stacking contexts */
  /* Has a ::backdrop pseudo-element */
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(4px);
}

Normal Document Layer

Top Layer
(dialog, fullscreen, popover)

Top layer elements render above everything, even elements with high z-index.

The overlay Property

/* Control top layer participation */

/* Default for dialogs/fullscreen */
dialog {
  overlay: auto;
  /* Stays in top layer when open */
}

/* Remove from top layer */
dialog {
  overlay: none;
  /* Behaves like normal positioned element */
  /* Uses regular z-index stacking */
}

/* Useful for animations */
dialog {
  overlay: auto;
  transition: 
    opacity 0.3s,
    display 0.3s allow-discrete,
    overlay 0.3s allow-discrete;
}

/* Closing animation */
dialog:not([open]) {
  opacity: 0;
  overlay: none;
}
/* Complete dialog example */
dialog {
  border: none;
  border-radius: 1rem;
  padding: 2rem;
  max-width: 500px;
  
  /* Top layer control */
  overlay: auto;
  
  /* Animations */
  opacity: 0;
  transform: scale(0.9);
  transition: 
    opacity 0.3s,
    transform 0.3s,
    display 0.3s allow-discrete,
    overlay 0.3s allow-discrete;
}

dialog[open] {
  opacity: 1;
  transform: scale(1);
}

/* Backdrop */
dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(8px);
  
  opacity: 0;
  transition: 
    opacity 0.3s,
    display 0.3s allow-discrete,
    overlay 0.3s allow-discrete;
}

dialog[open]::backdrop {
  opacity: 1;
}

Practical Example

<button onclick="dialog.showModal()">
  Open Modal
</button>

<dialog id="modal">
  <h2>Modal Title</h2>
  <p>Modal content here</p>
  <button onclick="modal.close()">
    Close
  </button>
</dialog>
#modal {
  border: none;
  border-radius: 1rem;
  padding: 2rem;
  box-shadow: 0 25px 50px rgba(0,0,0,0.2);
  
  /* Smooth entry/exit */
  overlay: auto;
  opacity: 0;
  transform: translateY(-20px);
  
  transition: 
    opacity 0.2s,
    transform 0.2s,
    display 0.2s allow-discrete,
    overlay 0.2s allow-discrete;
}

#modal[open] {
  opacity: 1;
  transform: translateY(0);
}

#modal::backdrop {
  background: rgba(0, 0, 0, 0.6);
  backdrop-filter: blur(4px);
}

Use Cases & Benefits

✅ Exit Animations

Keep element in top layer during closing animation, then remove it.

dialog:not([open]) {
  overlay: none; /* Removes from top layer after animation */
}

✅ Modal Stacking

Control which modals appear above others in the top layer.

dialog.background-modal {
  overlay: none; /* Use normal z-index */
  z-index: 100;
}

dialog.foreground-modal {
  overlay: auto; /* Top layer */
}

✅ Accessibility

Top layer ensures dialogs are always visible, avoiding z-index battles and ensuring screen readers can access them properly.
