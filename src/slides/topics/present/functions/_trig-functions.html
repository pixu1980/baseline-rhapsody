What are Trigonometric Functions?

CSS now supports mathematical functions for circular and angular calculations:

  • sin(), cos(), tan()
  • asin(), acos(), atan(), atan2()
  • pow(), sqrt(), hypot()
  • log(), exp()

Basic Example: Circular Motion

.circle-item {
  --angle: calc(var(--index) * 45deg);
  --radius: 150px;
  
  /* Position items in a circle */
  left: calc(50% + cos(var(--angle)) * var(--radius));
  top: calc(50% + sin(var(--angle)) * var(--radius));
  transform: translate(-50%, -50%);
}

/* 8 items around a circle */
.item:nth-child(1) { --index: 0; }
.item:nth-child(2) { --index: 1; }
.item:nth-child(3) { --index: 2; }
/* ... */

Smooth Wave Animation

@keyframes wave {
  from { --progress: 0; }
  to { --progress: 1; }
}

.wave-element {
  animation: wave 2s infinite linear;
  
  /* Create wave motion */
  transform: translateY(
    calc(sin(var(--progress) * 360deg) * 20px)
  );
}

/* Stagger multiple elements */
.wave-element:nth-child(2) {
  --phase: 0.2;
  transform: translateY(
    calc(sin((var(--progress) + var(--phase)) * 360deg) * 20px)
  );
}

Polar to Cartesian Conversion

/* Create a radar chart */
.radar-point {
  --angle: var(--data-angle);
  --value: var(--data-value); /* 0-1 range */
  --max-radius: 100px;
  
  --x: calc(cos(var(--angle)) * var(--value) * var(--max-radius));
  --y: calc(sin(var(--angle)) * var(--value) * var(--max-radius));
  
  left: calc(50% + var(--x));
  top: calc(50% + var(--y));
}

Advanced: Spiral Layout

.spiral-item {
  --index: var(--i);
  --total: 20;
  --progress: calc(var(--index) / var(--total));
  
  /* Spiral formula */
  --angle: calc(var(--progress) * 720deg); /* 2 full rotations */
  --radius: calc(var(--progress) * 200px); /* Expanding radius */
  
  left: calc(50% + cos(var(--angle)) * var(--radius));
  top: calc(50% + sin(var(--angle)) * var(--radius));
  
  /* Rotate item to follow spiral tangent */
  transform: translate(-50%, -50%) 
             rotate(calc(var(--angle) + 90deg));
}
