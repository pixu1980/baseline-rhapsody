Logical Properties ✅ Widely Available

/* Old way: physical directions */
.old {
  margin-left: 1rem;
  padding-right: 2rem;
  border-top: 1px solid;
}

/* New way: logical directions */
.new {
  margin-inline-start: 1rem;
  padding-inline-end: 2rem;
  border-block-start: 1px solid;
}

/* Works with RTL automatically! */
[dir="rtl"] .new {
  /* No changes needed! */
}
LTR: padding-inline-start
RTL: same property, flipped!
