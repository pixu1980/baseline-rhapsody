attr() - Read HTML Attributes

<!-- HTML -->
<a href="https://example.com" 
   data-tooltip="Visit our website">
  Link
</a>

<img src="photo.jpg" 
     alt="Sunset" 
     data-credit="Photo by Jane">
/* CSS */
a::after {
  content: " (" attr(href) ")";
  font-size: 0.875em;
  opacity: 0.7;
}

a[data-tooltip]:hover::before {
  content: attr(data-tooltip);
  position: absolute;
  background: #333;
  color: white;
  padding: 0.5rem;
  border-radius: 0.25rem;
  bottom: 100%;
}

img::after {
  content: attr(data-credit);
  display: block;
  font-size: 0.75em;
  color: #666;
}

counter() - Automatic Numbering

/* Define counters */
body {
  counter-reset: section;
}

h2 {
  counter-reset: subsection;
  counter-increment: section;
}

h2::before {
  content: counter(section) ". ";
  color: #667eea;
  font-weight: bold;
}

h3 {
  counter-increment: subsection;
}

h3::before {
  content: counter(section) "." counter(subsection) " ";
  color: #764ba2;
}

/* Nested counters */
ol {
  counter-reset: item;
  list-style: none;
}

ol li::before {
  counter-increment: item;
  content: counters(item, ".") ". ";
  color: #10b981;
}

symbols() - Custom List Styles

/* Cyclic symbols */
ul.emoji {
  list-style-type: symbols(cyclic "🎯" "✨" "🚀");
}

/* Numeric with custom symbols */
ol.steps {
  list-style-type: symbols(
    numeric 
    "➊" "➋" "➌" "➍" "➎"
  );
}

/* Alphabetic with symbols */
ol.letters {
  list-style-type: symbols(
    alphabetic 
    "Ⓐ" "Ⓑ" "Ⓒ" "Ⓓ"
  );
}

/* Fixed symbols (repeats last) */
ul.priority {
  list-style-type: symbols(
    fixed 
    "🔴" "🟡" "🟢"
  );
}

Custom List Markers:

  • First item gets 🎯
  • Second item gets ✨
  • Third item gets 🚀
  • Fourth item cycles back to 🎯

Practical Use Cases

📄 Print Stylesheet with Page Numbers

@media print {
  body {
    counter-reset: page;
  }
  
  @page {
    @bottom-right {
      content: "Page " counter(page);
    }
  }
}
