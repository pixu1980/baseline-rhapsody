What are CSS Filters?

Filters apply visual effects to elements without needing image editing:

  • blur() - Gaussian blur effect
  • brightness() - Adjust brightness
  • contrast() - Adjust contrast
  • grayscale() - Convert to grayscale
  • hue-rotate() - Rotate colors
  • saturate() - Adjust saturation
  • sepia() - Apply sepia tone
  • invert() - Invert colors
  • drop-shadow() - Add shadow

Basic Filter Examples

/* Blur effect */
.blurred {
  filter: blur(5px);
}

/* Brightness adjustment */
.bright {
  filter: brightness(150%);
}

.dark {
  filter: brightness(50%);
}

/* Convert to grayscale */
.grayscale {
  filter: grayscale(100%);
}

/* Combine multiple filters */
.vintage {
  filter: sepia(60%) contrast(110%) brightness(90%);
}

Backdrop Filter (Glassmorphism)

/* Blur background behind element */
.glass-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 12px;
}

/* macOS-style translucent sidebar */
.sidebar {
  backdrop-filter: blur(20px) brightness(110%);
  background: rgba(255, 255, 255, 0.7);
}

/* Dark glass effect */
.dark-glass {
  backdrop-filter: blur(16px) contrast(80%);
  background: rgba(0, 0, 0, 0.3);
}

Drop Shadow vs Box Shadow

/* box-shadow: follows the box */
.box-shadow {
  box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2);
}

/* drop-shadow: follows the element shape */
.drop-shadow {
  filter: drop-shadow(0 10px 20px rgba(0, 0, 0, 0.2));
}

/* Perfect for non-rectangular shapes */
.star {
  clip-path: polygon(
    50% 0%, 61% 35%, 98% 35%, 68% 57%, 
    79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%
  );
  filter: drop-shadow(0 5px 10px rgba(0, 0, 0, 0.3));
}

Hover Effects with Filters

/* Image hover effects */
.image-card img {
  filter: grayscale(100%) brightness(80%);
  transition: filter 0.3s ease;
}

.image-card:hover img {
  filter: grayscale(0%) brightness(100%);
}

/* Button glow effect */
.glow-button {
  filter: drop-shadow(0 0 0 transparent);
  transition: filter 0.3s ease;
}

.glow-button:hover {
  filter: drop-shadow(0 0 10px rgba(102, 126, 234, 0.8));
}

Dynamic Filters with Custom Properties

:root {
  --blur-amount: 0px;
  --brightness: 100%;
  --contrast: 100%;
}

.dynamic-filter {
  filter: 
    blur(var(--blur-amount))
    brightness(var(--brightness))
    contrast(var(--contrast));
  transition: filter 0.3s ease;
}

/* Change with JavaScript or media queries */
@media (prefers-reduced-motion: reduce) {
  :root {
    --blur-amount: 0px; /* Remove motion blur */
  }
}
